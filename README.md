# INSA_master
Conformal Prediction para Peso ao Nascer Quantificação de incerteza em redes neurais (PyTorch) prevendo peso ao nascer, comparando 3 métodos: PointNN+SC, QuantileNN+CQR (q05/q95) e Local SC. Duas calibrações (Global/Mondrian) e dois recortes de features (biomédico vs. metodológico), com α=0.10 (cobertura 90%) e avaliação por subgrupo.
